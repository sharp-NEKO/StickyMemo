アプリ名 StickyMEMO  
バージョン 1.0.0-R (安定)  
#### 特徴 ####
・CUIライク  
・ダブルクリックで背景色の変更が可能  
・マウスでドラッグ可能  
・フローティング表示  





### 使用方法 ###


メモする
→テキストを入力することでメモすることができます。

背景色を変える
→ダブルクリックをすると、色変更のウインドウが出てきます。そのウインドウで色を選んで、OKを押してください。

場所を変える
→マウスでドラッグしてください。

閉じる
→右下の✖を押してもらうことで、閉じることができます。

メモを２個以上出す
→StickyuMemo.exeを起動すれば、いくらでも表示可能です。
